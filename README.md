# Music

## Getting started

To run the app type:

```
$ yarn develop
```

and visit **http://localhost:1234**

### To run a production build type:

```
$ yarn build
```

### Run tests:

```
$ yarn test
```

### To run code formatting type:

```
$ yarn format
```

### For linting type:

```
$ yarn lint
```

> In order to run the above yarn commands you need to install
> the necessary modules first by running `$ yarn`
